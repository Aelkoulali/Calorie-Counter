# 🗂️ Calorie Counter 🔥
Calorie Counter is a modern web application built with HTML, CSS, and JavaScript that helps users calculate their daily calorie needs. Users fill out a form with their personal details, and the app validates the inputs to make sure everything is correct and meaningful. After validation, JavaScript performs the required calculations and updates the page instantly with the results—no page reload needed.

## ✅ Key Features
- Enter basic details (e.g., age/height/weight, gender, activity level).
- Validates the form input to ensure values are correct and complete.
- Calculates calories dynamically based on the user’s input.
- Updates the UI instantly to display results (like daily calorie needs or categories).
- Clean, modern layout using HTML + CSS with responsive styling.

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript :
    Required fields are not empty
    Numeric inputs are valid (numbers only)
    Values fall within reasonable ranges (example: weight > 0, height > 0)
   
## ▶️ How to Run
Open `index.html` in your browser.
